# EPFL_Auto_Chronos V1

This Javascript code helps fill the EPFL Chronos - a time keeping platform that records working hours per day.

## Features

+ Automatical fluctuation on working hours (uniformly distributed)
+ Assurance of minimal weekly working hours (> 41h)
+ Weekday detection

## Tutorial
1. First log in with you own Gaspar account from [SESAME](https://sesame.epfl.ch) and complete the MS authentication.
2. Open browser console (usually through *Inspection*), and paste the JS code.
3. Specify your _Gaspar No._, _Funding No._, _Funding Name_ in the script.
4. Specify the month you would like to submit.
5. Enter and done!

## Test Env.
The code has been tested in **EPFL Intranet** and **Cisco VPN** with **Chrome 146.0.7680.80 (Official Build) (64-bit)**.
